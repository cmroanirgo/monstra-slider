Slider Readme
================

Install the slider like any other plugin:
1. Go to Admin > Extends > Plugins and upload the zip
2. click on Install
3. In the Admin > Content section will be a new area 'Slider'

More documentation can be found at:
index.php?id=slider&action=usage
